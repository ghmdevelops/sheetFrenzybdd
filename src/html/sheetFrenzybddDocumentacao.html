Documentação SheetFrenzy BDD

Introdução

O SheetFrenzy BDD é uma ferramenta para otimizar o desenvolvimento de software utilizando o Desenvolvimento Orientado a Comportamento (BDD). Esta documentação detalha a funcionalidade de cada botão presente na página principal da ferramenta.

Botões da Barra Superior

Acessar Blog Compartilhado

Descrição: Este botão leva o usuário ao blog compartilhado, onde é possível encontrar artigos, tutoriais e outras informações úteis sobre BDD e desenvolvimento de software.

Direciona o usuário ao blog compartilhado, facilitando o acesso a recursos adicionais.
Fazer uma busca

Descrição: Ao clicar neste botão, o usuário pode realizar buscas dentro da página para encontrar informações específicas rapidamente.

Alterna a visibilidade do campo de busca, permitindo ao usuário exibir ou ocultar o campo conforme necessário.
Temas

Descrição: Permite ao usuário alternar entre diferentes temas de cores para a página, proporcionando uma experiência de visualização personalizada. 

Alterna entre diferentes temas visuais para a página.
Documentação

Descrição: Direciona o usuário para a documentação completa da ferramenta, onde ele pode encontrar guias de uso e referência.

Abre a documentação completa da ferramenta.
Conceitos

Descrição: Fornece acesso a conceitos básicos e explicações sobre BDD e a utilização da ferramenta.

Abre uma página ou modal com explicações sobre os conceitos básicos de BDD.
Editor Gherkin

Descrição: Abre o editor Gherkin, onde o usuário pode criar e editar cenários BDD usando a linguagem Gherkin.

Abre o editor Gherkin para criação e edição de cenários BDD.
Gravar uma página

Descrição: Inicia uma gravação da sessão atual, útil para criar tutoriais ou registrar processos.

Inicia a gravação da sessão atual do usuário.
Exportar para PDF

Descrição: Exporta a tabela ou o conteúdo atual da página para um arquivo PDF.

Exporta o conteúdo da página para um arquivo PDF.
Entrar em contato

Descrição: Permite ao usuário entrar em contato com o suporte ou a equipe de desenvolvimento.

Abre uma interface de contato para que o usuário possa se comunicar com o suporte.

Botões de Manipulação de Tabela

Diminuir número de linhas

Descrição: Diminui o número de linhas na tabela.

Diminui o número de linhas na tabela quando o botão é pressionado e interrompe o decremento quando o botão é solto.
Adicionar número de linhas

Descrição: Aumenta o número de linhas na tabela.

Aumenta o número de linhas na tabela quando o botão é pressionado e interrompe o incremento quando o botão é solto.
Adicionar Tabela

Descrição: Cria uma nova tabela com o número especificado de linhas e colunas.

Cria uma nova tabela com base nos valores de linhas e colunas fornecidos pelo usuário.
Apagar Linha

Descrição: Apaga a linha selecionada na tabela.

Apaga a linha selecionada na tabela.
Adicionar Linha

Descrição: Adiciona uma nova linha à tabela.

Adiciona uma nova linha à tabela.
Adicionar Coluna

Descrição: Adiciona uma nova coluna à tabela.

Adiciona uma nova coluna à tabela.
Baixar Excel

Descrição: Exporta a tabela para um arquivo Excel.

Exporta e baixa a tabela para um arquivo Excel.
Selecionar Excel

Descrição: Permite ao usuário importar dados de um arquivo Excel para a tabela.

Importa dados de um arquivo Excel selecionado pelo usuário e preenche a tabela com esses dados.

Botões Adicionais

Gerar Features

Descrição: Gera cenários de teste (features) para BDD com base na tabela atual.

Gera cenários de teste (features) com base na tabela atual.
Baixar Features

Descrição: Baixa os cenários de teste gerados em formato compatível com ferramentas BDD.

Exporta e baixa os cenários de teste selecionados pelo usuário.
Voltar para baixar

Descrição: Volta para a visualização anterior na interface de usuário.

Alterna entre diferentes estados de botões na interface.
Selecionar Todos

Descrição: Seleciona todos os cenários de teste para download ou edição.

Seleciona todos os cenários de teste na tabela.
Copiar para outras linhas

Descrição: Copia o conteúdo de uma célula para todas as linhas da tabela.

Copia o conteúdo de uma célula para todas as outras células na mesma coluna.
Gerar dashboard

Descrição: Gera um dashboard visual com gráficos baseados nos dados da tabela.

Gera um dashboard visual com gráficos baseados nos dados da tabela.
Gerar excel octane

Descrição: Converte a tabela para um formato específico de Excel. 

Converte a tabela para um formato específico de Excel.

Resumo das Funcionalidades

A página do SheetFrenzy BDD oferece uma série de funcionalidades para facilitar a criação e manipulação de cenários BDD. Os botões descritos acima permitem ao usuário manipular a tabela, gerar e baixar cenários de teste, personalizar a aparência da página e acessar informações e ajuda de maneira eficiente.

SHEETFRENZY BDD

Nosso site é dedicado a fornecer uma valiosa ferramenta para equipes que buscam uma maneira eficaz de criar, organizar e editar documentação de testes, promovendo uma colaboração mais eficiente e ágil no ciclo de desenvolvimento de software. Uma característica notável do site é a flexibilidade oferecida aos usuários, permitindo a edição ilimitada dos documentos gerados. Isso possibilita ajustes contínuos conforme as necessidades evoluem ou novos requisitos surgem. Com essa abordagem iterativa, o processo de desenvolvimento e teste se torna mais dinâmico e adaptável.
