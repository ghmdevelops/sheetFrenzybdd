Introdução

O SheetFrenzy BDD é uma ferramenta para otimizar o desenvolvimento de software utilizando o Desenvolvimento Orientado a Comportamento (BDD). Esta documentação detalha a funcionalidade de cada botão presente na página principal da ferramenta.

Estrutura HTML

Botões

  • Importar Excel: Permite ao usuário importar dados de um arquivo Excel para a tabela exibida na página.
    • Descrição: Este botão permite ao usuário importar dados de um arquivo Excel para a tabela exibida na página.
    • Funcionalidade: Ao clicar neste botão, o usuário pode selecionar um arquivo Excel de seu computador. O conteúdo do arquivo é lido e os dados são adicionados à tabela existente, mantendo qualquer informação já presente.
  • Salvar Tabela: Permite ao usuário salvar a tabela atual em um arquivo Excel.
    • Descrição: Este botão permite ao usuário salvar a tabela atual em um arquivo Excel.
    • Funcionalidade: Ao clicar neste botão, o usuário pode salvar os dados da tabela em um arquivo Excel no seu computador. A funcionalidade inclui a criação de um novo arquivo Excel com os dados atuais da tabela e a notificação do usuário sobre o sucesso da operação.
  • Adicionar Linha: Permite ao usuário adicionar uma nova linha à tabela.
    • Descrição: Este botão permite ao usuário adicionar uma nova linha à tabela.
    • Funcionalidade: Ao clicar neste botão, uma nova linha é adicionada ao final da tabela. Cada célula da nova linha contém um campo de entrada (input) onde o usuário pode inserir dados.
  • Excluir Linha: Permite ao usuário excluir a última linha da tabela.
    • Descrição: Este botão permite ao usuário excluir a última linha da tabela.
    • Funcionalidade: Ao clicar neste botão, a última linha da tabela é removida. Se houver apenas uma linha restante na tabela, uma mensagem de aviso é exibida indicando que não é possível excluir a última linha.
  • Adicionar Coluna: Permite ao usuário adicionar uma nova coluna à tabela.
    • Descrição: Este botão permite ao usuário adicionar uma nova coluna à tabela.
    • Funcionalidade: Ao clicar neste botão, o usuário é solicitado a fornecer o título da nova coluna. A nova coluna é então adicionada à tabela em uma posição especificada pelo usuário ou ao final da tabela.
  • Gerar Dashboard: Permite ao usuário gerar um dashboard com base nos dados da tabela.
    • Descrição: Este botão permite ao usuário gerar um dashboard com base nos dados da tabela.
    • Funcionalidade: Ao clicar neste botão, um dashboard é gerado mostrando a distribuição dos dados em diferentes categorias, como "Ok", "Nok", "Desplanejado", "Progredindo" e "Bug". O dashboard é exibido em um modal.
  • Copiar para Todos: Permite ao usuário copiar valores de uma célula para todas as células em uma coluna especificada.
    • Descrição: Este botão permite ao usuário copiar valores de uma célula para todas as células em uma coluna especificada.
    • Funcionalidade: Ao clicar neste botão, o usuário pode especificar a coluna e o intervalo de linhas onde deseja copiar o valor. O valor é então copiado para todas as células no intervalo especificado.
  • Filtro por CT: Permite ao usuário filtrar a tabela por um valor especificado no campo de filtro.
    • Descrição: Este botão permite ao usuário filtrar a tabela por um valor especificado no campo de filtro.
    • Funcionalidade: Ao clicar neste botão, a tabela é filtrada para mostrar apenas as linhas que contêm o valor especificado no campo de filtro. As linhas correspondentes são destacadas e a tabela é rolada para a primeira linha correspondente.
  • Tema: Permite ao usuário alterar o tema da página.
    • Descrição: Este botão permite ao usuário alterar o tema da página.
    • Funcionalidade: Ao clicar neste botão, uma janela de seleção de tema é aberta. O usuário pode escolher entre diferentes temas, como Claro, Escuro, Azul Oceano, entre outros. O tema selecionado é aplicado imediatamente à página.
  • Email: Permite ao usuário enviar um email com o conteúdo da tabela.
    • Descrição: Este botão permite ao usuário enviar um email com o conteúdo da tabela.
    • Funcionalidade: Ao clicar neste botão, uma janela é aberta para o usuário fornecer o endereço de email do destinatário e do remetente. Um email é então composto com os dados da tabela e enviado aos endereços especificados.
  • Print PDF: Permite ao usuário gerar um PDF do dashboard.
    • Descrição: Este botão permite ao usuário gerar um PDF do dashboard.
    • Funcionalidade: Ao clicar neste botão, o conteúdo do dashboard é convertido em um PDF, que pode ser baixado pelo usuário. O PDF contém uma captura de tela do dashboard, incluindo gráficos e resumos de dados.

Inputs

  • Upload de Arquivo: Campo de entrada para importar um arquivo Excel.
    • Descrição: Permite ao usuário selecionar um arquivo Excel de seu computador. O conteúdo do arquivo é lido e os dados são adicionados à tabela existente na página.
    • Funcionalidade: Permite ao usuário selecionar um arquivo Excel de seu computador. O conteúdo do arquivo é lido e os dados são adicionados à tabela existente na página.
  • Campo de Filtro: Campo de entrada para filtrar a tabela por CT.
    • Descrição: Permite ao usuário inserir um valor para filtrar a tabela.
    • Funcionalidade: Quando o valor é inserido e o botão de filtro é clicado, a tabela é filtrada para mostrar apenas as linhas que contêm o valor especificado.
  • Quantidade de Linhas: Campo para especificar a quantidade de linhas a serem adicionadas à tabela.
    • Descrição: Permite ao usuário definir o número de linhas a serem adicionadas à tabela ao criar uma nova tabela.
    • Funcionalidade: Permite ao usuário definir o número de linhas a serem adicionadas à tabela ao criar uma nova tabela.
  • Quantidade de Colunas: Campo para especificar a quantidade de colunas a serem adicionadas à tabela.
    • Descrição: Permite ao usuário definir o número de colunas a serem adicionadas à tabela ao criar uma nova tabela.
    • Funcionalidade: Permite ao usuário definir o número de colunas a serem adicionadas à tabela ao criar uma nova tabela.

Tabelas

  • Tabela de Dados: Tabela principal que armazena e exibe os dados importados e inseridos.
    • Descrição: Exibe os dados em formato tabular.
    • Funcionalidade: Cada célula da tabela contém um campo de entrada (input) onde o usuário pode inserir ou editar dados. A tabela pode ser manipulada através dos botões para adicionar/excluir linhas e colunas, e pode ser filtrada e ordenada conforme necessário.

Modais

  • Modal Principal: Modal que exibe o dashboard gerado com base nos dados da tabela.
    • Descrição: Exibe um resumo visual dos dados da tabela, incluindo gráficos e resumos.
    • Funcionalidade: O modal pode ser aberto ao clicar no botão "Gerar Dashboard" e pode ser fechado pelo usuário após a visualização.

Funcionalidades JavaScript

Eventos de Duplo Clique

  • Descrição: Abre um modal de edição ao dar um duplo clique em um campo de texto (input[type="text"]) dentro da tabela.
  • Funcionalidade: Ao dar um duplo clique em uma célula da tabela, um modal é aberto permitindo ao usuário editar o conteúdo da célula em um campo de texto maior.

Importação de Excel

  • Descrição: Importa dados de um arquivo Excel selecionado pelo usuário e adiciona esses dados à tabela existente.
  • Funcionalidade: O usuário seleciona um arquivo Excel de seu computador. O conteúdo do arquivo é lido e os dados são adicionados à tabela existente. Uma notificação é exibida informando o sucesso da importação.

Salvamento de Tabela

  • Descrição: Salva a tabela atual em um arquivo Excel.
  • Funcionalidade: Os dados da tabela são exportados para um novo arquivo Excel que é salvo no computador do usuário. Uma notificação é exibida informando o sucesso da operação.

Gerenciamento de Tabela

  • Adicionar Linha: Adiciona uma nova linha à tabela.
    • Descrição: Adiciona uma nova linha à tabela.
    • Funcionalidade: Uma nova linha é adicionada ao final da tabela, com campos de entrada vazios para que o usuário possa inserir novos dados.
  • Excluir Linha: Exclui a última linha da tabela.
    • Descrição: Exclui a última linha da tabela.
    • Funcionalidade: A última linha da tabela é removida. Se houver apenas uma linha restante, uma mensagem de aviso é exibida.
  • Adicionar Coluna: Adiciona uma nova coluna à tabela.
    • Descrição: Adiciona uma nova coluna à tabela.
    • Funcionalidade: O usuário é solicitado a fornecer o título da nova coluna, que é então adicionada à tabela na posição especificada.

Reconhecimento de Voz

  • Descrição: Permite ao usuário preencher os campos da tabela usando comandos de voz.
  • Funcionalidade: O reconhecimento de voz é ativado, permitindo que o usuário insira dados na tabela através de comandos de voz. Os dados inseridos são preenchidos automaticamente nos campos de entrada da tabela.

Configuração de Estilo

  • Descrição: Permite ao usuário alterar o estilo da tabela e dos campos de entrada.
  • Funcionalidade: O usuário pode selecionar diferentes cores de fundo, cores de fonte e fontes para personalizar a aparência da tabela. As alterações são aplicadas imediatamente aos campos de entrada da tabela.

Filtragem e Ordenação

  • Filtragem: Filtra a tabela por um valor especificado.
    • Descrição: Filtra a tabela por um valor especificado.
    • Funcionalidade: O usuário insere um valor no campo de filtro e a tabela é atualizada para mostrar apenas as linhas que contêm o valor especificado.
  • Ordenação: Ordena a tabela por uma coluna especificada.
    • Descrição: Ordena a tabela por uma coluna especificada.
    • Funcionalidade: Ao clicar no cabeçalho de uma coluna, a tabela é ordenada em ordem crescente ou decrescente com base nos valores da coluna.

Copiar para Todos

  • Descrição: Copia valores de uma célula para todas as células em uma coluna especificada.
  • Funcionalidade: O usuário especifica a coluna e o intervalo de linhas onde deseja copiar o valor. O valor é então copiado para todas as células no intervalo especificado.

Dashboard

  • Descrição: Gera um dashboard com base nos dados da tabela.
  • Funcionalidade: Um resumo visual dos dados da tabela é gerado, incluindo gráficos e resumos. O dashboard é exibido em um modal, permitindo ao usuário visualizar a distribuição dos dados em diferentes categorias.

Tema

  • Descrição: Permite ao usuário alterar o tema da página.
  • Funcionalidade: O usuário pode escolher entre diferentes temas, como Claro, Escuro, Azul Oceano, entre outros. O tema selecionado é aplicado imediatamente à página.

Print e Download

  • Print PDF: Gera um PDF do dashboard.
    • Descrição: Gera um PDF do dashboard.
    • Funcionalidade: O conteúdo do dashboard é convertido em um PDF que pode ser baixado pelo usuário. O PDF contém uma captura de tela do dashboard, incluindo gráficos e resumos de dados.
  • Download Excel: Permite ao usuário baixar a tabela como um arquivo Excel.
    • Descrição: Permite ao usuário baixar a tabela como um arquivo Excel.
    • Funcionalidade: A tabela é exportada como um arquivo Excel e baixada para o computador do usuário.
SHEETFRENZY BDD

Nosso site é dedicado a fornecer uma valiosa ferramenta para equipes que buscam uma maneira eficaz de criar, organizar e editar documentação de testes, promovendo uma colaboração mais eficiente e ágil no ciclo de desenvolvimento de software. Uma característica notável do site é a flexibilidade oferecida aos usuários, permitindo a edição ilimitada dos documentos gerados. Isso possibilita ajustes contínuos conforme as necessidades evoluem ou novos requisitos surgem. Com essa abordagem iterativa, o processo de desenvolvimento e teste se torna mais dinâmico e adaptável.
