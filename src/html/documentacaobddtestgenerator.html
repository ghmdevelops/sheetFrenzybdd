SheetFrenzy BDD - Documentação

Botão "Buscar"

Ação: Ativa a função `Buscar` quando clicado.

Função: Essa função alternará a visibilidade de algum elemento na página, como um campo de pesquisa ou resultados.

Seletor de Tema (themeSelector)

Ação: Ativa uma ação relacionada à mudança de tema e permite selecionar o tema desejado a partir de uma lista suspensa.

Função: Pode alternar entre temas de luz, escuro ou outros temas personalizados com base na implementação do código, proporcionando uma experiência visual personalizável para o usuário.

Opções de Tema:

  • Tema Claro: Define um tema claro para a página, ideal para ambientes com boa iluminação.
  • Tema Escuro: Estabelece um tema escuro, adequado para ambientes de baixa luminosidade ou preferência do usuário.
  • Tema Azul Oceano: Aplica um tema com tonalidades azuis inspiradas no oceano, proporcionando uma estética única.
  • Tema Azul Escuro: Utiliza um esquema de cores azul escuro, ideal para uma aparência elegante e moderna.
  • Tema Noite: Adota um tema noturno, suave para os olhos durante a noite.
  • Tema Dia: Aplica um tema claro e vibrante, ideal para ambientes diurnos.

Botões para Manipulação de Tabela

Incrementar/Decrementar Linhas e Colunas: Modificam o número de linhas e colunas da tabela, permitindo uma personalização flexível da estrutura da tabela.

Criar Tabela: Inicia a criação da tabela com base nos valores especificados para linhas e colunas, facilitando a geração rápida de tabelas personalizadas.

Manipulação de Linhas e Colunas: Adiciona, remove linhas e adiciona colunas à tabela, oferecendo controle granular sobre a estrutura da tabela.

Botão "Exportar Excel" (Baixar Excel)

Ação: Inicia o processo de exportação da tabela para um arquivo Excel.

Função Esperada: Exportar os dados da tabela para um arquivo Excel, proporcionando uma maneira conveniente de compartilhar ou salvar os dados em um formato amplamente utilizado.

Botão "Selecionar Arquivo"

Ação: Ativa a seleção de um arquivo para importação.

Função: Permitir que o usuário selecione um arquivo para ser importado para a página, facilitando a incorporação de dados externos.

Botão "Copiar para Todos"

Ação: Copia o conteúdo para todas as células da tabela.

Função: Deve copiar o conteúdo da célula atual e aplicá-lo a todas as células da tabela, oferecendo uma maneira eficiente de replicar dados.

Botão "Dashboard"

Ação: Abre um modal para exibir um dashboard com gráficos e informações.

Função: Apresentar visualizações gráficas e estatísticas em um modal interativo, proporcionando uma visão rápida e compreensível dos dados da tabela.

Botão "Projetos BDD"

Ação: Abre um pop-up para visualização de projetos BDD.

Função: Exibi informações sobre projetos BDD de forma clara e acessível dentro de um pop-up, facilitando o acesso a detalhes importantes.

SHEETFRENZY BDD

Nosso site e dedicado a fornecer uma valiosa ferramenta para equipes que buscam uma maneira eficaz de criar, organizar e editar documentação de testes, promovendo uma colaboração mais eficiente e ágil no ciclo de desenvolvimento de software. Uma característica notável do site é a flexibilidade oferecida aos usuários, permitindo a edição ilimitada dos documentos gerados. Isso possibilita ajustes contínuos conforme as necessidades evoluem ou novos requisitos surgem. Com essa abordagem iterativa, o processo de desenvolvimento e teste se torna mais dinâmico e adaptável.
